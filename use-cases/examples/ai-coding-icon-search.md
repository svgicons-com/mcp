# AI Coding Icon Search

This is a documentation example, not a real customer submission.

## Problem

A developer wants a low-risk first MCP workflow before letting an AI client edit code.

## Solution

Use metadata-only icon search, compare options, inspect one selected icon, and review results before requesting any code change.

## Tools Used

- `search_icons`
- `search_icon_sets`
- `get_icon`

## Prompts Used

- [Search icons](../../prompts/search-icons.md)

## Workflow

1. Search for five candidates.
2. Compare icon sets.
3. Inspect one selected icon.
4. Review metadata and visual fit.

## Review Checklist

- Several options inspected.
- Visual fit verified.
- Metadata reviewed where relevant.
- No generated code accepted blindly.

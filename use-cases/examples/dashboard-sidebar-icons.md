# Dashboard Sidebar Icons

This is a documentation example, not a real customer submission.

## Problem

A dashboard sidebar needs icons for users, billing, analytics, settings, and support.

## Solution

Use MCP recommendations and search to compare candidate icons and icon sets before editing code.

## Tools Used

- `recommend_icons_for_ui`
- `search_icons`
- `search_icon_sets`

## Prompts Used

- [Dashboard navigation](../../prompts/dashboard-navigation.md)
- [Design-system review](../../prompts/design-system-icon-review.md)

## Workflow

1. Ask for recommendations by navigation label.
2. Compare visual style across candidates.
3. Reject duplicate or ambiguous icons.
4. Review naming before adding icons to code.

## Review Checklist

- Visual consistency.
- Semantic fit.
- Duplicates and ambiguity.
- Team naming.
- No tokens or private data included.

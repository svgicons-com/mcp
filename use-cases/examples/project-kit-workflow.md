# Project Kit Workflow

This is a documentation example, not a real customer submission.

## Problem

A project needs a reusable Icon Collection for dashboard screens.

## Solution

Use MCP to search and recommend icons, generate a collection if the account supports it, then review before export.

## Tools Used

- `search_icons`
- `recommend_icons_for_ui`
- `generate_icon_kit_for_project`
- `export_icon_collection`

## Prompts Used

- [Project Kit workflow](../../prompts/project-kit-workflow.md)

## Workflow

1. Search and recommend icons.
2. Generate a collection from the project brief.
3. Review selected icons and naming.
4. Export only after approval.

## Review Checklist

- Account permissions checked.
- Selected icons reviewed.
- Generated files reviewed.
- No tokens or private data included.
